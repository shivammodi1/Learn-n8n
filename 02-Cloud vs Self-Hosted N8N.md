# Cloud Based n8n
- It is run and managed by the n8n team.
- Official hosted version of n8n(means you don't have to worry about server maintenance, updates, or backups).
- No setup required, just sign up and start using it.
- Comes with built-in features like automatic updates, backups, and scalability.
- But it is cost associated with it (subscription plans available).
- Not ideal for sensitive data or private workflows due to data being stored on n8n's servers.

# Self-Hosted n8n
- In self-hosted n8n, We run n8n on own server or system.
- Full control over data, workflows, and configurations.
- Can be hosted on VPS(Virtual Private Server),cloud , or local machine.
- Open-source version of n8n, free to use.
- Requires setup and maintenance (server management, updates, backups).
- Ideal for sensitive data or private workflows as data is stored on own infrastructure.
# How to Install Self-Hosted n8n
```bash
npm install n8n -g # Install n8n globally using npm
n8n start # Start the n8n server
```
